# Bước 2: Yêu Cầu Chức Năng

## 1. Functional Requirements

### 1.1 Format Yêu Cầu

Mỗi yêu cầu theo format:
```
FR-{module}-{số}: {Mô tả}
- Ưu tiên: P1/P2/P3
- Acceptance Criteria: GIVEN-WHEN-THEN
```

### FR-{ID}: {Tên}

